# EmoTracker

EmoTracker is an item, location, and progress tracker for randomized games
(such as A Link to the Past Randomizer). It provides a customizable overlay
of items, maps, and hints that updates as you play, and supports a wide
variety of community-built game packs.

This organization is the community-maintained home for EmoTracker and its
supporting projects.

## Repositories

- **[EmoTracker](https://github.com/EmoTracker-Community/EmoTracker)** —
  The tracker application itself. A desktop client (originally WPF, with an
  in-progress Avalonia port for cross-platform support) that loads game
  packs and tracks items, locations, and progression in real time.

- **[EmoTracker-Service](https://github.com/EmoTracker-Community/EmoTracker-Service)** —
  The package and update service that backs the tracker. Hosts the game
  pack repository, installer assets, supported-games metadata, and
  application version manifests consumed by the EmoTracker client.
